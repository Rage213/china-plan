# План на Китай

Мини-приложение для расчета накоплений на учебу в Китае.

## Что умеет

- считает цель в юанях и рублях;
- автоматически подтягивает курс юаня к рублю;
- показывает, сколько нужно откладывать в месяц;
- сравнивает план с доходом от вайбкодинга;
- хранит чеклист первых заказов;
- сохраняет план в браузере.

## Как открыть

Открой `index.html` в браузере.
